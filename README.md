## Installation

```
$ sudo npm install -g sb-game
```

### Run game - Even

```
$ brain-even
```

### Run game - Calc

```
$ brain-calc
```

### Run game - Find GCD

```
$ brain-gcd
```

### Run game - Progression

```
$ brain-progression
```

### Run game - Prime

```
$ brain-prime
```
